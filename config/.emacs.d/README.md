# Emacs 設定

## キーバインド

### 基本

| キー | 機能 |
|------|------|
| `C-h` | backspace |
| `C-?` | ヘルプ |

### ファイル検索

| キー | 機能 |
|------|------|
| `C-c p` | プロジェクト内ファイル検索 (git) |
| `C-c C-p` | 全ファイル検索 (.gitignore 含む) |

### 検索・移動

| キー | 機能 |
|------|------|
| `C-s` | バッファ内検索 (consult-line) |
| `C-x b` | バッファ切り替え |
| `M-g g` | 行番号ジャンプ |
| `M-s r` | ripgrep 検索 |

### LSP (Go)

| キー | 機能 |
|------|------|
| `M-.` | 定義へジャンプ |
| `M-,` | 戻る |
| `C-c C-r` | リネーム |
| `C-c C-a` | コードアクション |

### Org-mode

#### 見出し操作

| キー | 機能 |
|------|------|
| `TAB` | 見出し折りたたみトグル |
| `S-TAB` | 全体の折りたたみトグル |
| `M-RET` | 新しい見出し/リスト項目 |
| `M-LEFT/RIGHT` | 見出しレベル変更 |
| `M-UP/DOWN` | 見出し移動 |

#### TODO

| キー | 機能 |
|------|------|
| `C-c C-t` | TODO 状態切り替え |
| `S-LEFT/RIGHT` | TODO 状態変更 |
| `C-c C-s` | スケジュール設定 |
| `C-c C-d` | 締切設定 |

#### チェックボックス

| キー | 機能 |
|------|------|
| `C-c C-c` | チェックボックストグル |
| `M-S-RET` | チェックボックス付き項目追加 |

#### リンク・エクスポート

| キー | 機能 |
|------|------|
| `C-c C-l` | リンク挿入 |
| `C-c C-o` | リンクを開く |
| `C-c C-e` | エクスポート |

#### その他

| キー | 機能 |
|------|------|
| `C-c C-c` | コードブロック実行 / 表再計算 |
| `C-c '` | コードブロック編集 |
| `C-c C-x C-i` | タイマー開始 |
| `C-c C-x C-o` | タイマー停止 |

## 機能

- **自動保存**: 1秒ごとに自動保存
- **eglot + gopls**: Go ファイルで自動起動
- **vertico + orderless**: fuzzy 補完
- **which-key**: キーバインドヒント表示

## 依存

```bash
# gopls
go install golang.org/x/tools/gopls@latest

# fd (ファイル検索用)
brew install fd

# ripgrep (grep 検索用)
brew install ripgrep
```
