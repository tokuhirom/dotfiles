# Zellij Configuration

## Keybindings (tmux-like)

prefix は `C-t`（Ctrl+t）

### Pane Navigation

| Key | Action |
|-----|--------|
| `C-t h` | 左のペーンへ移動 |
| `C-t j` | 下のペーンへ移動 |
| `C-t k` | 上のペーンへ移動 |
| `C-t l` | 右のペーンへ移動 |

### Pane Operations

| Key | Action |
|-----|--------|
| `C-t -` | 下に分割（水平） |
| `C-t "` | 下に分割（水平） |
| `C-t \|` | 右に分割（垂直） |
| `C-t %` | 右に分割（垂直） |
| `C-t x` | ペーンを閉じる |
| `C-t z` | フルスクリーン切替 |
| `C-t f` | フローティングペーン切替 |

### Tab Operations

| Key | Action |
|-----|--------|
| `C-t c` | 新しいタブ |
| `C-t n` | 次のタブ |
| `C-t p` | 前のタブ |
| `C-t 1-9` | タブ番号で移動 |

### Other

| Key | Action |
|-----|--------|
| `C-t [` | スクロールモード |
| `C-t r` | リサイズモード |
| `C-t d` | デタッチ |
| `C-t w` | セッションマネージャ |
| `C-t C-t` | キャンセル（locked に戻る） |
| `Esc` | キャンセル（locked に戻る） |

### Resize Mode (`C-t r`)

| Key | Action |
|-----|--------|
| `h/j/k/l` | サイズ増加（左/下/上/右） |
| `H/J/K/L` | サイズ減少 |
| `+/=` | 全体的に増加 |
| `-` | 全体的に減少 |
| `Enter/Esc` | リサイズモード終了 |

### Scroll Mode (`C-t [`)

| Key | Action |
|-----|--------|
| `j/k` | 1行スクロール |
| `d/u` | 半ページスクロール |
| `C-f/C-b` | 1ページスクロール |
| `f` | 検索モード |
| `e` | エディタで編集 |
| `C-c` | スクロールモード終了 |

## zj コマンド

カレントディレクトリ名でセッションを作成/アタッチ:

```bash
$ zj
```

## Alt ショートカット（常時有効）

| Key | Action |
|-----|--------|
| `Alt h/l` | タブ/ペーン移動 |
| `Alt j/k` | ペーン移動 |
| `Alt n` | 新しいペーン |
| `Alt f` | フローティング切替 |
| `Alt +/-` | リサイズ |
